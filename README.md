# miniproject
hi, this is just some instructions on how to run this program.

1. type pip install -r requirements.txt to install all packages.
if you are having trouble with installing packages, you can install them manually by typing (pip install package_name) in terminal.  eg. (pip install streamlit)

2. type myenv/Scripts/activate to activate the virtual environment.

3. type streamlit run app.py to run the program. this might take a while.

4. open the .env file and copy the api key or you can click the "Get API Key" and create a new gemini API key.

5. after the previous step, you can upload you pdf(s) and hit the "Submit and Process" to upload the pdf(s) and process them.

6. DONE!
